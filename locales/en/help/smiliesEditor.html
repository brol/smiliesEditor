Initialization:
If the current theme does not contain the directory smilies where are the smilies stored in theory, you can create it by clicking the button initialize.
New smiley:
To create a new smiley, you associate a string, for example: :-), the image of your choice.
Image:

Image formats accepted by the form are JPG or PNG. The longest side must be shorter than 24 pixels.
You can also send a valid ZIP file (containing a file smilies.txt).

Toolbar:
It is composed of displayed smilies: .

Form and comments

Preview:
If the option is checked, the images associated with smilies codes will be automatically displayed.
Text form:
You can set the text preceding the toolbar. If you leave the text blank, then the word smilies is positioned in the field.

Writing posts

Smilies in toolbar:
If you want to use smilies in the writing of new posts, you must activate the option in the user preferences (the option is off by default).
